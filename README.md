# フォルダーホーク
フォルダーホーク (Folder Hawk) Is a File Scanning Application to help Track Down Big Files.

Unlike Window's (no help at all), Folder Hawk Is 100% Open Source & Free. Its Features:

- Trace Files in Folder
- Lists Files from Size (High to Low)
- Delete Folder/Files


> [!WARNING]
> Common Bugs:
> Freezing apon Big Folders,
> Freezing when Deleting Files 1TB+
